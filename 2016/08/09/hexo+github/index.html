【学无止境】使用Hexo+Github来建立自己的博客

  • Android

Hexo搭建

环境配置

1.安装Node。
2.安装git。
3.安装hexo

初始化完成后cd <folder>执行npm install ,来安装相应的插件，具体可以看<folder>目录下的package.json 。

执行步骤

1.进入<folder>中，运行 hexo generate，生成静态页面。
2.执行hexo server，部署服务器之后就可以看到效果了,http://localhost:4000(CTRL+C 停止服务器）。
3.如果能看到效果说明hexo已经可以正常使用，hexo还提供了许多主题，点击一个主题会发现其实是github上的开源项目，你只需要进入目录下的themes，执行 git clone <git上的链接> 即可获取对应的主题，例如：https://github.com/DrakeLeung/hexo-theme-again.git。然后打开根目录下的_config.yml文件，修改theme,theme: again（具体值看README)。

常见问题

1. 执行hexo deploy时报出ERROR Deployer not found: git 错误。

遇到这个问题是因为没有安装hexo的git插件，执行npm install hexo-deployer-git --save。默认配置下，只需要在博客目录下执行npm install就会安装目录下package.json配置的插件。

2. 执行hexo deploy时报出fatal: could not read Username for GitHub错误。

遇到这个问题一般是因为在window下得cmd上操作，可以尝试使用git bash或者cygwin下进行，这样执行hexo deploy后，它才会要求你输入github的账号密码。

3. 执行hexo generate时报出YAMLException: can not read a block mapping entry; a multiline key may not be an implicit key at line 4, column 1错误。

遇到这个错误一般是要发表的文章的格式错误注意里面的title: 、 tag: 和description: 冒号后面都是需要空格再加上内容的。
